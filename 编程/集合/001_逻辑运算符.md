逻辑运算符

&和&&：当且仅当 a、b 都为 true 时，结果为 true ；

\| 和 \|\| ： 当且仅当 a、b 都为 false 时，结果为 false ；

\^ : a 与 b 相异时，结果为 true ；

注意：逻辑运算符操作的都是boolean类型的变量，而且结果也是boolean类型。

![](001_逻辑运算符_000.png)

 

& 与 && 的区别：& 和 \| 左右两边的式子一定会执行（比较笨），&& 和 \|\| 只要左边的式子能得出结果，右边的式子就不会执行（比较聪明）。
