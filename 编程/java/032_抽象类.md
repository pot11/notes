抽象类，强制子类重写抽象的方法

 

 

问题：

1.父类成员变量要私有吗？

答：可以私有，但是必须提供外界访问方式get/set

 

2.抽象类可以有构造方法吗？

答：是，可以有构造方法，但是不能直接通过构造方法创建对象

 

 

[设计思想：保证继承体系功能的完整性]{.mark}
