**通常情况下，只有post请求方式才会使用到请求体，请求体中都是用户表单提交的数据，每一项数据都使用键值对**

**key=value，多组值使用&相连。**

**例如；username=tom&password=1234**

**浏览器默认把请求与响应的格式做好了，不用操心，只需要enter就行**
