```json
模版id：yy_order_0000001
标题：售后申请
场景：申请售后
群消息：
{订单号}订单申请售后，请尽快处理
{orderId}订单申请售后，请尽快处理
跳转：售后单列表

发送方式：
{
    "qiYeWxParam": {
        "isAvailable": true,
        "atUserList": "",
        "webhookUrl": ""
    }
}
变量参数：
[
    {
        "codeDesc": "订单号",
        "code": "orderId"
    }
]
```
