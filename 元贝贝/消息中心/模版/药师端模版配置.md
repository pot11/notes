# 药师端模版配置

```json
通用发送方式模版
{
    "appPushParam": {
        "isAvailable": true
    },
    "appInBoxParam": {
        "isAvailable": true,
        "appInBoxType": "2"
    }
}
```

```json
模版id：ys_prescription_0000001
标题：审方任务
场景：医生开具处方
普通站内信内容：/push模版：
{医生姓名}医生于{开方时间}开具了一张{药方类型}，请尽快完成审核！

{doctorName}医生于{openPrescriptionTime}开具了一张{medicinePrescriptionType}，请尽快完成审核！

跳转：处方详情页

发送方式：
{
    "appPushParam": {
        "isAvailable": true
    },
    "appInBoxParam": {
        "isAvailable": true,
        "appInBoxType": "2"
    }
}
变量参数：
[
    {
        "codeDesc": "医生姓名",
        "code": "doctorName"
    },
    {
        "codeDesc": "开方时间",
        "code": "openPrescriptionTime",
    },
    {
        "codeDesc": "药方类型",
        "code": "medicinePrescriptionType",
    }
]
```
