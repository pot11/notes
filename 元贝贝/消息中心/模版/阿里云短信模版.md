# 阿里云短信模版

## 患者端

```json
模版id：pt_inquiry_0000003
模版标题：通话提醒
场景：语音、视频问诊预约提前5分钟通知
（点击开始咨询时，出现申请小程序订阅消息）
模版类型：通知短信
变量属性：
就诊人：thing2
医生姓名：thing4
预约服务：thing5
预约时间：time3
短链编码：shortCode

模版内容：
{thing2}有一个{thing4}医生的{thing5}将会在{time3}开始，请点击点击https://hospital-test.ybbhealth.com/api/admin/appRedirect/{shortCode}，进入问诊诊室等待医生发起邀请

```

```json
模版id：pt_member_0000001
标题：会员到期
场景：会员到期前7天
模版类型：通知短信
变量属性：
患者姓名：patientName
套餐名称：packageName
套餐类型：packageType
天数：days
模版内容：
{patientName}的{packageName}{packageType}即将在{days}天后过期，请及时关注权益使用情况

```

```json
模版id：pt_question_0000001
标题：问卷调查
场景：医生发送问卷给患者

变量属性：

{thing4}医生提醒{thing1}{thing7}，点击https://hospital-test.ybbhealth.com/api/admin/appRedirect/{shortCode}查看详情
```
