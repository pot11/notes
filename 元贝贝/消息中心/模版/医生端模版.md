================会员==============

```json
模版id：dc_member_0000001
标题：就诊人绑定
场景：
套餐绑定了就诊人

普通站内信内容：
{注册手机号}用户购买了您的{会员名称+规格}已经绑定就诊人为{患者姓名}
{registphone}用户购买了您的{packageNameAndSpu}已经绑定就诊人为{patientName}
短信模版内容:
{注册手机号}用户购买了您的{会员名称+规格}已经绑定就诊人为{患者姓名}
{registphone}用户购买了您的{packageNameAndSpu}已经绑定就诊人为{patientName

发送方式：
{
    "appInBoxParam": {
        "isAvailable": true,
        "appInBoxType": "3"
    },
    "smsParam": {
        "isAvailable": true,
        "smsChannel": "1",
        "smsType": "1",
        "templateId": "SMS_488835102"
    }
}
变量参数：
[
    {
        "codeDesc": "注册手机号",
        "code": "registphone",
        "smsCode": "registphone"
    },
    {
        "codeDesc": "会员名称+规格",
        "code": "packageNameAndSpu",
        "smsCode": "packageNameAndSpu"
    },
    {
        "codeDesc": "患者姓名",
        "code": "patientName",
        "smsCode": "patientName"
    }
]
```
